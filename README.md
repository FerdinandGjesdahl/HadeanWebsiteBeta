# Hadean Ventures Website

This project is a static website. There is no build step or dependency install.

## Run locally

From the repo root:

```bash
python3 serve_local.py
```

Then open the URL printed in the terminal, usually:

```text
http://127.0.0.1:4173
```

If port `4173` is already in use, the script automatically selects the next free port.
